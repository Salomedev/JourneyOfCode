# Nails by Ale

A website for a nail and braids stylist.  
Clients can:
- View available services
- Browse a photo gallery
- Book appointments online
- Contact directly via WhatsApp

## 📂 Project Details
This project includes:
- **index.html**: main structure of the site
- **styles.css**: design and visual styles
- **app.js**: booking logic and WhatsApp integration
- **images/**: gallery photos of nail art and braids

## 🚀 Technologies
- HTML
- CSS
- JavaScript

## 🎯 Purpose
To provide clients with an easy way to explore services, see past work, and schedule appointments online.
